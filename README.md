### 1. 💾 Типы данных 

#### Задача 1: Калькулятор сложных процентов
Разработка функции для расчета и форматирования вывода накоплений по годам с учетом сложного процента.
* **Входные данные:** `initial_deposit` (начальный вклад), `years` (срок в годах), `interest_rate` (годовая ставка).
* **Особенности:** Округление сумм до 2 знаков после запятой, форматированный строковый вывод.
* **Пример вывода** *(для вклада 1000₽, 3 года, ставка 10%)*:
  ```text
  Год 1: 1100.00 руб.
  Год 2: 1210.00 руб.
  Год 3: 1331.00 руб.
  ```
🔗 **Решение:** [Task1.cs](CollegePractice/DataType/Task1.cs)

#### Задача 2: Отрисовка ромба
Написание функции для вывода в консоль ромба (диаманта) из символов `X` заданного размера.
* **Условия:** Длина диагонали равна `N` (нечётное число), центр ромба должен быть пустым (пробел).
* **Пример вывода** *(для N=5)*:
  ```text
    X  
   X X 
  X   X
   X X 
    X  
  ```
🔗 **Решение:** [Task2.cs](CollegePractice/DataType/Task2.cs)

### 2. 📐 ООП: Геометрические фигуры

**Тема:** Прямоугольники (Rectangle)
Реализация класса `Rectangle`, инкапсулирующего состояние и поведение геометрической фигуры.
