### 1. Типы данных

**Задание 1.** Написать функцию, которая формирует и возвращает строку с расчетом сложных процентов по годам.

**Функция принимает:**
- `initial_deposit` — начальный вклад (положительное число)
- `years` — количество лет (положительное целое число)
- `interest_rate` — годовая процентная ставка (положительное число)

**Функция должна возвращать** строку, где для каждого года расчета указана сумма накоплений, округленная до 2 знаков после запятой.

**Пример** возвращаемого значения для `initial_deposit=1000`, `years=3`, `interest_rate=10`:
```text
Год 1: 1100.00 руб.
Год 2: 1210.00 руб.
Год 3: 1331.00 руб.
```
